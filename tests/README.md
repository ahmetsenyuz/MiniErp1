# Tests

This directory contains all test files for the MiniErp1 application.

## Directory Structure

- `unit/` - Unit tests for individual modules
- `integration/` - Integration tests for API endpoints
- `e2e/` - End-to-end tests

## Running Tests

To run all tests:
```bash
npm test
```

To run tests in watch mode:
```bash
npm run test:watch
```
