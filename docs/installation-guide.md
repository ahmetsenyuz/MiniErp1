# Installation Guide

This document provides step-by-step instructions for installing and setting up the Mini-ERP system.

## Prerequisites

Before installing the Mini-ERP system, ensure the following prerequisites are met:

- Node.js (version 14 or higher)
- npm (version 6 or higher)
- MongoDB (version 4.4 or higher)

## Installation Steps

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the application: `npm start`

## Verification

After installation, verify that the application is running correctly by accessing the following URL in your browser:
http://localhost:3000

## Troubleshooting

If you encounter any issues during installation, please refer to the troubleshooting guide in this documentation package.
