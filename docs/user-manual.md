# User Manual

This manual provides guidance on using all functionalities of the Mini-ERP system.

## Overview

The Mini-ERP system is designed to manage business operations including inventory tracking, sales orders, purchases, and supplier management. It offers a web-based interface for users to interact with the system.

## Getting Started

1. Access the application through your web browser at http://localhost:3000
2. Log in using your credentials
3. Navigate through the different modules using the sidebar menu

## Core Functionalities

### Inventory Management
- View current stock levels
- Add new products to inventory
- Update product information
- Manage product categories

### Sales Orders
- Create new sales orders
- View existing sales orders
- Update order status
- Generate sales reports

### Purchases
- Create purchase orders
- Track incoming shipments
- Manage supplier relationships
- Process payments

### Supplier Management
- Add new suppliers
- View supplier details
- Update supplier information
- Track supplier performance

## Navigation Guide

The system features a responsive navigation menu that allows users to access different modules:

1. Dashboard - Main overview page
2. Products - Inventory management
3. Sales Orders - Order processing
4. Purchases - Purchase management
5. Suppliers - Supplier information
6. Reports - System reports and analytics

## User Roles and Permissions

The system supports different user roles with varying permissions:

- Administrator: Full access to all features
- Manager: Access to most features except user management
- Staff: Limited access to specific modules

## Support and Help

For assistance with the system, please contact your system administrator or refer to the online help resources.
