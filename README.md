# MiniErp1

A Mini-ERP system for managing business operations.

## Overview

This project is a lightweight Enterprise Resource Planning system designed to handle core business processes including inventory management, sales, and procurement.

## Features

- Inventory tracking
- Sales order processing
- Procurement management
- User roles and permissions
- Reporting capabilities

## Getting Started

These instructions will get you a copy of the project up and running on your local machine.

### Prerequisites

- Node.js (v14 or higher)
- npm (v6 or higher)

### Installation

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the application: `npm start`
